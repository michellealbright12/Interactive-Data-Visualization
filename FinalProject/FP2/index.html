Bar Chart

from https://bl.ocks.org/mbostock/3887051

A bar chart could show the distributions of lakes across the three regions. It could also be used to show distribution of lakes across counties within a state.


Stacked Bar Chart

from https://bl.ocks.org/mbostock/3886208

A stacked bar chart can show the value of multiple attributes for each lake. We could use this on a subset of the data, given that there are 500,000 something data points. For example, it could show how chemical pollution varies by lake or on average over lakes in a state. The stacked element would provide more information as to what type of chemicals are polluting (NH4, NO2, etc.)


Choropleth

from http://ssz.fr/parite/

A choropleth is a map in which different regions are shaded to illustrate a particular attribute. We only have data on lakes from certain regions of the US, but our data does include the latitude and longitude of the lake so that would make plotting it on a map easier. Then we would just have to add in a visualization for the attribute that we would want to examine.


Histogram

from https://bl.ocks.org/mbostock/b2fee5dae98555cf78c9e4c5074b87c3


A histogram shows a continuous range of values, and each bar represents a "bin" of values. We could use this to show the range of photo user days across a state or region. Or to tie it back to the lake's chemical data, it could show the range of chemicals concentrations across all lakes'


Scatterplot

from https://bost.ocks.org/mike/nations/

A scatterplot could show the relationship between lake size and depth. The color of each dot or the size of each dot could correlate to a lake's location (state) or to another independent variable, like the land development levels near the lake.
